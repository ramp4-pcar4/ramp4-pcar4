updateQuery

  • updateQuery(queryMapUpdate): string

  • Updates the query part of the url with passed in values.

    For example:

    • orginal url: http://example?flag=red&demohell=true
    • queryMapUpdate: { flag: undefined, demohell: false, acid: cat }
    • resulting url: http://example?demohell=false&acid=cat

    Parameters

    • queryMapUpdate: UrlQueryMap

      an object of values to be added or replaced on the query of the url; if any values are undefined, their corresponding keys will be removed from the query.

    Returns string

    updated url
