RAMP 4 Samples Catalogue

Classic Samples

Samples from development which were ported to the Samples suite.
  • 01. Main Sample / Default Sample. A classic mess of different services and legend structures. Some notable things include visibility sets, disabled visibility controls, custom icon stacks. Numerous individual tweaks are on various layers. Feature LayerMap Image LayerWFS LayerPointsPolygonsExclusive SetInfo Sections
  • 02. Lots of Panels. A number of panels open at startup. This includes some of the original "externally defined" panels which do trivial interactions. WFS LayerPointsCustom Fixtures
  • 03. Mercator Basemap Only. Contains only the map, in Mercator projection (satellite view). No fixtures or map elements are added. No appbar, no panels, etc. Nothing!
  • 04. Lambert Basemap Only. Contains only the map, in Lambert projection (topographic view). Most UI elements are removed, except for the appbar (empty) and overview map. Nothing!
  • 05. Map Image Layer. Contains a legend with two intial map image layers. One layer is configured so that it is not visible on the map on load. Map Image Layer
  • 06. WMS Layer. Contains a legend with an initial collection of WMS layers. WMS Layer
  • 07. Custom Renderer. Contains a legend with a collection of layers that have optional renderer configurations applied to them. Map Image Layer
  • 08. Custom Overview Map. Contains only the overview map. The overview map uses a custom basemap, as well as other configured settings (border width, colour, etc.).
  • 09. Custom Appbar Button. Contains only a custom appbar button.
  • 10. Custom North Arrow. Contains only a custom north arrow.
  • 11. Grid. Contains a legend with a collection of unique layers. Each layer has a datagrid. Datatable
  • 17. Upgraded RAMP2 Config. An instance generated by passing in a RAMP2 config through the config upgrader. Nothing!
  • 18. Layers in error state. The main sample, but with some of the layers in error state.
  • 19. Layers with disabled options. Layers with various options (datatable, settings, etc.) disabled. Feature Layer Points
  • 20. Layer with metadata. A layer with metadata. Feature Layer Points
  • 21. Section Items. Legend section items with different settings (exclusive, hidden, etc.). Exclusive Set Feature Layer Points
  • 26. Fog Hilighter. A demonstration of feature highlighting with the fog hilighter. Feature Highlighting
  • 28. Layers with exclusive fields enabled . Layers with with a subset of their fields specified in the config.
  • 29. Grid config. A layer with a customized datatable. Datatable
  • 30. Map Image Layers with groups. Map image layers with groups specified in the config. Map Image Layer
  • 31. Custom symbology stack. A legend with a custom configured legend symbology stack. Symbology Stack
  • 32. Offscale Layer. A demonstration of what happens when you add an offscale layer to RAMP. Offscale Layer
  • 33. Delayed Start. A sample that loads the map and layers after a delay of a few seconds. Offscale Layer

Simple Samples

Samples with no fancy configuration. Useful for testing the basics.
  • 12. Geosearch. Contains only the geolocation search panel.
  • 13. Help. Contains only the help panel.
  • 14. Mapnav. Contains only the mapnav.
  • 15. Feature Layers. Two feature layers. Feature LayerPoints
  • 16. Map Image Layers. One map image layer with two children. Map Image LayerPoints
  • 22. Info Sections. All basic info section types Info Sections
  • 27. File Layers. One CSV layer, one shapefile layer, and one GeoJson layer. File Layers
