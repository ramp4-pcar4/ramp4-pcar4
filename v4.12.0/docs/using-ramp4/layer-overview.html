Layer Configuration Properties ​

Layer configuration objects define the layers on the map. They typically exist in the layers array of an instance configuration object, or are provided at runtime using the LayerAPI.

Below is a list of the various properties that can be specified for a layer config. There are also configurations for Map Image Sublayer and WMS Sublayer definitions.
